

# 人口行为模型项目


## 背景
这个项目是上家公司承接的某个国企的基层治理相关的一个8千万的项目，过了1年了，前两天整理了一下上传用于学习。

## 项目概述

本项目旨在通过HDBSCAN聚类算法分析人口行为模式。项目包括数据生成、数据预处理、特征工程、模型训练和结果评估等步骤。
